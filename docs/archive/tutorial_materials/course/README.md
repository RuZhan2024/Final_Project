# Course Index

This directory contains the instructor-led rebuild course for the fall detection repository.

## Delivery model

- The course is markdown-only.
- Lesson documents are the source of truth.
- Students copy code from the lessons into `course_project/`.
- There is no separate bundled teaching codebase.
- There is no competing tutorial implementation path outside `docs/course/`.

## Course structure

- `docs/course/week01/`
- `docs/course/week02/`
- `docs/course/week03/`
- `docs/course/week04/`
- `docs/course/week05/`

Each week contains:

- `README.md`
- `day01.md`
- `day02.md`
- `day03.md`
- `day04.md`
- `day05.md`
- `day06.md`
- `day07.md`

## Current status

- Week 1 is complete and sets the quality/style baseline.
- Week 2 is the next active build stage and starts from the window artifacts created in Week 1.

## How to use this course

- Start at `docs/course/week01/README.md`.
- Work day by day.
- Type the code from each lesson into the matching `course_project/` path.
- Treat the lesson markdown as the canonical teaching implementation.
