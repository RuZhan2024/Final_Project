# Archive Docs

This directory stores historical documentation that is no longer the primary source of truth.

## Contents

- `audit_completed/`
  - legacy audit and planning documents retained for traceability.
- `reports/`
  - archived report plans and retired notes that are no longer part of the active report surface.
- `training/`
  - older training-focused notes and upgrade task documents kept for reference only.

## Usage Rule

- Do not use archived files as the main reference for current status or final submission.
- For current status, use:
  - `docs/reports/`
  - `docs/project_targets/`
