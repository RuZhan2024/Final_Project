Archive policy for frozen reporting artifacts.

This directory stores superseded but intentionally retained evidence that should not be treated as the current authoritative report layer.

Subdirectories:
- `replay_matrix_legacy_20260402/`: earlier replay matrices and pre-fix summary files retained for audit history
- `replay_runtime_iterations_20260402/`: intermediate replay/runtime fix iterations retained for diagnosis, not for final reporting

Authoritative live files remain in:
- `artifacts/reports/`
- `artifacts/figures/report/`

Diagnostic-only files live in:
- `artifacts/reports/diagnostic/`
- `artifacts/figures/report/diagnostic/`
